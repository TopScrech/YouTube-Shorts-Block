# Focus mode for YouTube
Hide YouTube shorts & other distractions

<img width="683" height="512" alt="0x0ss" src="https://github.com/user-attachments/assets/cd523e9d-8a60-4451-8ce2-80fdf8a75ece" />

## Features
- Hide shorts in the nav bar, home page and in channel sections
- Hide likes/dislikes
- Hide likes count
- Return to playlist button when watching a video form playlist
- Hide subscriber count (macOS-only)

## Supported Platforms
- iOS / iPadOS 15+
- macOS 12+
